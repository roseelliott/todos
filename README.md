## todos
=====

TODOs - Backbone/Dust client side template with server REST api and based on http://documentcloud.github.com/backbone/examples/todos

## Live Demo 
[DEMO](http://meebike.com:3000/)

## Required modules:

* node
*	express
* consolidate
*	underscore
*	backbone
*	dustjs-linkedin
*	jquery
* iscroll

### Compile dust templates

$ ./compile

### Run server
$ node server
